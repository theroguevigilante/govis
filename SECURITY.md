# Security Policy

### Supported Versions

The latest version of Govis is the only actively supported version.

### Reporting a Vulnerability

Please do not open a public GitHub issue for security vulnerabilities.
Instead, report them by GitHub's private vulnerability report or email at:
- [naimish@naimish.xyz](mailto:naimish@naimish.xyz)
- [naimish82@proton.me](mailto:naimish82@proton.me).

If possible include:

- Steps to reproduce
- Logs

I will acknowledge reports as soon as possible.

### Disclaimer

This project is intended for research, education, and experimentation.
Although care has been taken during implementation, govis has not yet undergone an independent security audit.
Use caution before deploying it in production.

### Disclosure

Please allow the maintainers reasonable time for fixes before publicly disclosing vulnerabilities.
